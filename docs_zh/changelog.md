# 更新日志

升级到最新版本：

=== "Linux/macOS"

    ```
    $ pip3 install -U apiflask
    ```

=== "Windows"

    ```
    > pip install -U apiflask
    ```

或者先检查当前安装的版本：

=== "Linux/macOS"

    ```
    $ pip3 show apiflask
    ```

=== "Windows"

    ```
    > pip show apiflask
    ```
